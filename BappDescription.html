This extension allows penetration testers to create detailed vulnerability reports. The reports are available in both PDF and Word formats for ultimate client convenience.

Features

  • Create vulnerability reports based on OWASP Top 10 2021 categories.
  • Support for both PDF and Word report formats.
  • Customizable vulnerability details, including the description of the vulnerability, its impact, evidence including request/response transactions, and customizable remediation plans. It even provides options to include screenshot evidence.
  • Risk level classification for understanding the level of threat associated with each vulnerability.
  • Easy-to-use interface for hassle-free interaction.
  • Support for embedding images in reports for more comprehensive detail.

Usage

  1. Right-click on any request in Burp Suite.
  2. Select Add to Pentest Report from the context menu.
  3. Fill in the vulnerability details as prompted by the tool.
  4. Add screenshots or evidence if needed, to back up your claims.
  5. Click Add to Report to include the compiled vulnerability into the report.
  6. Generate the final report in your preferred format (PDF or Word).
